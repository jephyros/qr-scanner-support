## 한국어 (Korean)

### 앱 이름
Server Monitor — 서버 상태 감시

### 부제 (Subtitle, 30자 이내)
서버·API 상태를 한눈에

### 설명 (Description)

서버와 API를 스마트폰에서 한눈에 모니터링하세요.

여러 서버의 상태를 주기적으로 자동 체크하고, 이상이 감지되면 즉시 푸시 알림으로 알려드립니다. 별도 계정이나 외부 서비스 없이, 앱 하나로 내 서버를 직접 감시하세요.

【주요 기능】

• 서버 URL 등록 및 상태 모니터링
HTTP GET / POST / PUT / DELETE / PATCH 메서드 지원. 여러 서버를 한 화면에서 관리하세요.

• 세밀한 성공 조건 설정
HTTP 상태코드(200, 201 등)와 JSON 응답값(특정 키 일치·포함·미만 등) 두 가지 조건을 조합해 정확한 성공 기준을 정의할 수 있습니다.

• 다양한 인증 방식 지원
Bearer Token, API Key, Basic 인증, 커스텀 헤더 — 인증이 필요한 API도 바로 모니터링하세요.

• 백그라운드 자동 체크
앱을 닫아도 주기적으로 서버 상태를 확인합니다. 체크 간격은 항목별로 1분, 5분, 10분, 30분, 60분 중 선택 가능합니다.

• 이상 감지 즉시 Push 알림
서버가 다운되거나 예상한 응답이 오지 않으면 바로 알림을 보내드립니다. 복구 시에도 알림을 받을 수 있습니다.

• 공인 IP 제한
특정 공인 IP에서만 체크가 실행되도록 설정할 수 있습니다. 사무실 내부망 서버 등 접근 제한이 있는 환경에서 유용합니다.

• 서버 설정 백업 및 복원
전체 서버 설정을 JSON 파일로 내보내고 불러올 수 있습니다. 기기를 교체할 때도 설정을 그대로 이어 사용하세요. 인증 토큰 포함 여부를 선택해 보안을 유지할 수 있습니다.

• 한국어 / 영어 지원

## English

### App Name
Server Monitor — Uptime & Alerts

### Subtitle (30 chars)
Monitor servers & APIs instantly

### Description

Keep your servers and APIs under watch — right from your iPhone.

Server Monitor automatically checks the health of your servers on a schedule and sends you an instant push notification the moment something goes wrong. No accounts, no external services — your data stays on your device.

【Key Features】

• Monitor multiple server URLs
Supports HTTP GET, POST, PUT, DELETE, and PATCH. Manage all your endpoints in one place.

• Flexible success conditions
Define success by HTTP status code (200, 201, etc.), JSON response value (equals, contains, less than, and more), or both combined — so you get alerted only when it really matters.

• Full authentication support
Bearer Token, API Key, Basic Auth, and Custom Headers — monitor authenticated APIs with no extra setup.

• Background auto-check
Checks continue even when the app is closed. Set individual check intervals per server: 1, 5, 10, 30, or 60 minutes.

• Instant push notifications
Get notified immediately when a server goes down or returns an unexpected response. Receive a recovery alert when it comes back online.

• IP restriction
Restrict monitoring to a specific public IP address — useful for internal network servers or office-only environments.

• Backup & restore settings
Export all server configurations to a JSON file and import them on a new device. Choose whether to include auth tokens when exporting.

• Korean / English language support

【Perfect for】
- Developers managing multiple servers and API endpoints
- Operations and DevOps teams who need fast incident response
- Anyone who wants lightweight server monitoring without extra tools

All data is stored locally on your device and never sent to any external server.
